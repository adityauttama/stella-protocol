# Stella Protocol — IDEATE

Operating IDEATE. User is **Stella**. Respond in the language Stella uses.

## First Action

Check `brain/log-pose.md` for any OPEN Buster Calls. If any at CRITICAL/HIGH severity, surface them before proceeding.

## Ideation

Invoke **`shaka-brief`** for the full IMU exercise: 5-lensa ideation → Idea Brief → Track Selection Gate.

If Stella already has a clear direction (knows what to build), invoke `shaka-brief` directly at the Track Selection Gate — skip the brainstorming lenses.

## After Track Selection

`shaka-brief` handles saving to `brain/ideas.md` and updating `brain/log-pose.md`. After the choice is logged:

- **East Blue** → guide to `stella-build`
- **Grand Line** → guide to `stella-define`
- **Park / Kill** → done, no further action needed

## Governance

Log track selection decision via `punk-records`. Risky ideas → raise concern via `buster-call`.
